Huella Verde Base de Datos

Web application built with Laravel, managing a PostgreSQL database. It enables users to perform CRUD operations on products, including image management via Cloudinary through a dedicated API.

Huella Verde React

Web page developed with React and Bootstrap, facilitating the visualization of products loaded with "Huella Verde Base de Datos", featuring authentication through Auth0 to allow interactions between users through comments.

Huella Verde API

Node.js API REST that offers various endpoints and services, seamlessly integrating authentication with Auth0

GymTech

GymTech analyzes videos of gym exercises, identifying key body points to track movement over time. It calculates data such as speed, acceleration, energy expenditure, and approximate calories burned, helping users gain insights into their workout performance.

Recoder

Recoder is a Python app that encodes, decodes, and visualizes text, video, and images. Using a custom algorithm and base64 encoding, it securely transforms media for safe storage and transmission. Recoder also allows easy visualization of encoded content.

Quick Notes

Quick Notes is a simple and convenient Chrome extension that allows you to create, save, and manage multiple notes directly within your browser. The extension features a user-friendly interface that resembles post-it notes, making it easy to jot down quick thoughts, reminders, or tasks.
